# かんたんインストール

Substance Painter で塗った内容を、Maya にリアルタイム反映するツールです。
**この手順どおりに進めれば、数分で導入できます。**

インストールは **Maya 側** と **Substance Painter 側** の2つだけです。

---

## 事前準備（1分）

以下の3ファイルを、**同じフォルダ**にまとめてダウンロードしておきます。

- `install.py` … Maya 用インストーラー
- `maya_live_sync.py` … Maya 本体
- `sp_live_sync_plugin.py` … Substance Painter 本体
- `sp_to_aiStanderdSurface.py` … aiSS（自動テクスチャ割り当て機能）本体

> `install.py` と `maya_live_sync.py`  `sp_to_aistanderdsurface.py`は**必ず同じフォルダ**に置いてください（インストーラーが隣のファイルを探します）。

---

## ① Maya 側（ドラッグ&ドロップするだけ）

1. Maya を起動します。
2. エクスプローラーから **`install.py`** を、Maya の **3D画面（ビューポート）** に**ドラッグ&ドロップ**します。
3. 「インストールが完了しました」と出れば終了です。

これだけで、以下が自動で行われます。

- 本体を Maya の所定フォルダへコピー
- シェルフに **`LiveSync`** ボタンを追加
- Maya 起動時に自動で読み込むよう登録

**起動方法**：シェルフの **`LiveSync`** ボタンを押すだけです。

> ボタンが出ない場合は、Maya のスクリプトエディタ（Python）で次を実行しても起動できます。
> ```python
> import maya_live_sync
> maya_live_sync.show_ui()
> ```

---

## ② Substance Painter 側（ファイルを1つ置くだけ）

1. Substance Painter を起動します。
2. 上部メニュー **Python > Plugins Folder** をクリックします（フォルダが開きます）。
3. 開いたフォルダに **`sp_live_sync_plugin.py`** をコピーします。
4. Substance Painter を**一度再起動**します。
5. 上部メニュー **Python > sp_live_sync_plugin** に**チェック**を入れて有効化します。
6. 初回だけ**設定ウィザード**が自動で開きます。
   → **すべて初期値のまま進めてOK**です（Maya 側と自動で一致します）。

---

## ③ 使ってみる

1. **Maya**：`LiveSync` ボタンでウィンドウを開き、**監視を開始**します。
2. **SP**：プロジェクトを開き、Live Sync パネルで**同期を開始**します。
3. SP で少し塗ると、数秒後に Maya 側へ反映されます。

---

## ⚠️ 凹凸（Height / ディスプレイスメント）を使うときだけ必要な設定

Height（変位）を使うと、箱などが**歪む・膨らむ・球になる**ことがあります。
これは正常な仕様で、**メッシュ側に1つ設定を足す**だけで直ります。

対象メッシュを選び、アトリビュートエディタの **shape ノード > Arnold > Subdivision** で：

- **Type** を `linear`（箱・平面）または `catclark`（岩・布などの有機物）に変更
- **Iterations** を `2`〜`3` に設定

> それでも全体が膨らむ場合は、同じ shape の **Displacement Attributes > Zero Value** を `0.5` にしてください。

---

## うまくいかないときは

| 症状 | 対処 |
| --- | --- |
| Maya にドロップしても反応しない | `install.py` と `maya_live_sync.py` が同じフォルダにあるか確認 |
| SP にメニューが出ない | ファイルを置いた後、SP を再起動したか確認 |
| Maya に反映されない | Maya 側で「監視」、SP 側で「同期」の両方が開始されているか確認 |
| 凹凸が歪む・膨らむ | 上の「Height を使うときの設定」を参照 |
|同期されない|テクスチャをaiSSでfinalもしくはliveフォルダのテクスチャを割り当ててるか確認|
|同期されない(1)|SPとMayaが同じプロジェクト・テクスチャを開いてなければ同期されません。同じプロジェクト・テクスチャを開いているか確認してください。|
|同期されない(2)|表示品質を何度も切り替えてください|
## 同期を有効にしているのにテクスチャが反映されないときは

以下の手順を行ってみてください。
1. **Maya**を一度再起動し、AIStanderdSurfaceを`HyperShade`から全てのノードを削除する
2. **SP**でCtrl+Sでプロジェクトを保存する
3. **Maya**で`AiSS`を起動し`C/MayaLiveSync/Final/あなたのプロジェクト名_ハッシュ値`を指定し、スキャン/割り当てをしてください
4. **Maya**で`LiveSyncパネル`で表示品質を切り替えてください

上記の手順でもテクスチャが反映されない時はご連絡ください。
