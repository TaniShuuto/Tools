# かんたんインストール

Substance Painter で塗った内容を、Maya にリアルタイム反映するツールです。
**この手順どおりに進めれば、数分で導入できます。**

インストールは **Maya 側** と **Substance Painter 側** の2つだけです。

> このページに書かれているファイル名・ボタン名・画面の文言は、実際のツールと
> 一言一句合わせてあります。表示されているものと違う場合は、古いバージョンの
> ファイルを使っている可能性があるため、最新版をダウンロードし直してください。

---

## 事前準備（1分）

以下のファイルを、**すべて同じフォルダ**にまとめてダウンロードしておきます。
ファイル名は大文字・小文字・スペルまで実際のファイルと完全に一致させてください
（1文字でも違うと動作しません）。

| ファイル名 | 役割 |
| --- | --- |
| `install.py` | Maya 用インストーラー |
| `maya_live_sync.py` | Maya 本体（必須） |
| `sp_live_sync_plugin.py` | Substance Painter 本体（必須） |
| `sp_to_aiStandardSurface.py` | aiSS（テクスチャ自動割り当て機能）本体（任意） |
| `udim_setup.py` | UDIM自動セットアップ機能本体（任意） |
| `udim_setup_icon.png` | UDIM機能のシェルフアイコン（任意、無くても動作します） |

> `sp_to_aiStandardSurface.py` と `udim_setup.py` は無くてもインストールでき、
> その場合は該当する機能だけが導入されません。後から追加する場合は、
> ファイルを同じフォルダに置いて `install.py` を再度ドラッグ&ドロップすれば
> 追加分だけ導入されます。

---

## ① Maya 側（ドラッグ&ドロップするだけ）

1. Maya を起動します。
2. エクスプローラーから **`install.py`** を、Maya の **3D画面（ビューポート）** に**ドラッグ&ドロップ**します。
3. 「SP -> Maya Live Sync のインストールが完了しました。」というダイアログが出れば終了です。

これだけで、以下が自動で行われます。

- 本体を Maya の所定フォルダへコピー
- 同じフォルダに置いたファイルに応じて、シェルフに以下のボタンを追加
  - **`LiveSync`**（必須。maya_live_sync.py がある場合）
  - **`aiSS`**（任意。sp_to_aiStandardSurface.py がある場合）
  - **`UDIM`**（任意。udim_setup.py がある場合）
- Maya 起動時に自動で読み込むよう登録

完了ダイアログには **[バージョン一覧]** も表示されるので、今どのバージョンが
入っているかその場で確認できます。

**起動方法**：シェルフの **`LiveSync`** ボタンを押すだけです。

> ボタンが出ない場合は、Maya のスクリプトエディタ（Python）で次を実行しても起動できます。
> ```python
> import maya_live_sync
> maya_live_sync.show_ui()
> ```

### tips：もし既にインストール済みの場合

以下の手順を踏むことで確実に動作します。

1. `uninstall.py` をドラッグ&ドロップします。
2. シェルフにある起動ボタン（`LiveSync` / `aiSS` / `UDIM`）を右クリック →
   `Delete` から削除します。
3. **通常のインストール手順（上記①）を行ってください。**

---

## ② Substance Painter 側（ファイルを1つ置くだけ）

1. Substance Painter を起動します。
2. 上部メニュー **Python > Plugins Folder** をクリックします（フォルダが開きます）。
3. 開いたフォルダに **`sp_live_sync_plugin.py`** をコピーします。
4. Substance Painter を**一度再起動**します。
5. 上部メニュー **Python > sp_live_sync_plugin** に**チェック**を入れて有効化します。
6. 初回だけ**セットアップウィザード**が自動で開きます。
   → **すべて既定値のまま「完了」まで進めてOK**です（Maya 側と自動で一致します）。

パネルを開くと、一番上に太字の **「Live Sync を有効にする」** というチェックボックスがあります。これをONにすると同期が始まります。

---

## ③ aiSS（テクスチャの自動割り当て）の使い方

**aiSS** は、SPでエクスポートしたテクスチャをスキャンして自動的に
aiStandardSurfaceマテリアルとして作成する機能です（`sp_to_aiStandardSurface.py`
を導入した場合のみ使えます）。

1. シェルフの **`aiSS`** ボタンを押して起動します。
2. ウィンドウが開くので、上部の **「テクスチャフォルダ」** にエクスポートした
   テクスチャのフォルダパスを指定します。
3. **「フォルダをスキャンしてテクスチャセットを検出」** ボタンを押すと、
   検出したテクスチャセットの一覧が表示されます。
4. 一覧の中で、マテリアルとして作成したいセットにチェックを入れます。
5. 下部の **「チェックしたセットのマテリアルを作成」** ボタンでマテリアルを
   作成します。

---

## ④ UDIM自動セットアップの使い方

**UDIM自動セットアップ**は、UDIMタイル分割されたテクスチャを検出して
自動でArnold用マテリアルをセットアップする機能です（`udim_setup.py` を
導入した場合のみ使えます）。

1. シェルフの **`UDIM`** ボタンを押して起動します。
2. **「テクスチャフォルダ」** にUDIMテクスチャのフォルダパスを指定します
   （SP側のLive Sync設定から自動で検出できた場合は最初から入力されています）。
3. **「スキャン実行」** ボタンを押すと、検出したモデル・UDIMタイル構成が
   一覧表示されます。
4. セットアップしたいモデルにチェックを入れ、**「選択モデルをセットアップ」**
   ボタンで自動接続します。

## ⑤ 実際に使ってみる（Live Syncの基本フロー）

1. モデルにaiStandardSurfaceマテリアルを割り当てた状態でMayaからエクスポート
2. SP側でインポート
3. SP側で **「Live Sync を有効にする」** をON
4. Maya側で **`LiveSync`** ウィンドウを開く
   - 「状態」欄の「プロジェクト連携」がもし未設定なら、**「SPプロジェクトを設定」**
     ボタンを押してリンクします
5. Maya側で **「監視（自動反映）: OFF」** ボタンを押してONにする
6. SP側で塗った内容が、Maya側へ自動的に反映されます

### tips：表示品質について

Maya側の **「表示品質: プレビュー(リアルタイム)」** ボタンがOFF（プレビュー）
の間は、テクスチャの画質が低い状態で表示されます。高画質にするには、この
ボタンを押して **「表示品質: 高画質(Final)」** に切り替えてください。

高画質モードの場合、SP側で**保存（Ctrl+S）を行ったときにのみ**Maya側へ
反映されます（プレビューのようにリアルタイムには反映されません）。

## うまくいかないときは

| 症状 | 対処 |
| --- | --- |
| Maya にドロップしても反応しない | `install.py` と `maya_live_sync.py` が同じフォルダにあるか確認してください |
| SP にメニューが出ない | ファイルを置いた後、SP を再起動したか確認してください |
| Maya に反映されない | Maya 側で「監視」、SP 側で「Live Sync を有効にする」の両方がONになっているか確認してください |
| 凹凸が歪む・膨らむ | Height/Displacementの接続はremapValueで自動補正されています。それでも歪む場合はメッシュのSubdivision設定を確認してください |
| 同期されない | aiSSまたはUDIM機能で、Final（高画質）フォルダ側のテクスチャを割り当てているか確認してください |
| 同期されない（プロジェクトが違う） | SPとMayaで同じプロジェクト・同じシーンを開いているか確認してください。プロジェクトが一致していないと同期されません |
| 同期されない（表示品質切替後） | 表示品質ボタン（プレビュー⇔高画質）を一度ONOFFし直してみてください |

### 同期を有効にしているのにテクスチャが反映されないときは

以下の手順を順番に試してください。

1. **Maya**を一度再起動し、`HyperShade` から aiStandardSurface 関連のノードを
   一旦すべて削除する
2. **SP**で `Ctrl+S` を押してプロジェクトを保存する
3. **Maya**で `aiSS` ボタンを起動し、
   `C:/SPMayaLiveSync/final/<プロジェクト名>_<英数字6桁>/` フォルダ
   （プロジェクト名部分はSPのファイル名、末尾6桁はプロジェクトごとに
   自動生成される識別用の文字列です）を指定してスキャン・割り当てをする
4. **Maya**の `LiveSync` ウィンドウで表示品質ボタンを押し直す

上記の手順でもテクスチャが反映されない場合は、`26au0119@jec.ac.jp` までご連絡ください。
